Для корректной работы программы необходимо:
1) Установить все необходимые библиотеки 
Команда: pip install -r requirements.txt
2) Запустить файл main.py
3) После запуска выберете, подходящий для вас режим работы программы и задайте координаты первых трех точек 
4) После будут открываться окна с изображениями:
	а) Куба в пространстве, проеции на плоскость XY
	б) Тетраэдра в пространстве, проеции на плоскость XY
	в) Четырёхугольной пирамиды в пространстве, проеции на плоскость XY
Чтобы перейти к следующему изображению необходимо закрыть окно с уже имеющейся на экране фигурой

5)Также после просмотра всех фигур - в корневой папке создастся 3 файла, в которых находятся 3 объемные фигуры.
Данные изображения можно открыть с помощью Point3D, для проверки правильности построения самой фигуры, а не ее расположения в пространстве.
